Article 1 — Objet et champ d'application

Art. 1

Les présentes Conditions Générales d'Utilisation (CGU) régissent l'utilisation de l'application PerfCNC, outil d'analyse de performance pour ateliers d'usinage CNC. En utilisant PerfCNC, vous acceptez les présentes CGU dans leur intégralité.

PerfCNC est proposé en deux versions : une version gratuite avec collecte anonymisée de statistiques agrégées, et une version Pro avec confidentialité totale des données.

🔓

Article 2 — Traitement des données — Version gratuite

Art. 2

En version gratuite, après chaque analyse, les statistiques agrégées suivantes sont sauvegardées localement à des fins d'amélioration du service :

  • Nombre total de saisies (entier)
  • Moyenne des heures produites (valeur numérique)
  • Total des pièces KO (entier)
  • Répartition des causes d'arrêt (dictionnaire cause → nombre d'occurrences)
  • Répartition par famille de pièces (dictionnaire famille → nombre de saisies)
  • TRS calculé, si disponible (pourcentage)
✓ Ce qui n'est JAMAIS collecté : nom de fichier · dates réelles · références pièces · codes équipe · données brutes ligne par ligne · toute information permettant d'identifier votre atelier ou vos clients.

Chaque enregistrement est associé à un identifiant aléatoire unique (hash SHA-256 d'un sel aléatoire) qui ne permet aucune corrélation entre plusieurs analyses d'un même utilisateur.

🔒

Article 3 — Traitement des données — Version Pro

Art. 3

En version Pro (USE_CLAUDE_API = True), aucune donnée n'est collectée par PerfCNC. Les fichiers Excel sont traités en mémoire et supprimés immédiatement après analyse. Aucun fichier temporaire n'est conservé sur le serveur.

Les données sont transmises à l'API Anthropic (Claude) uniquement sous forme de statistiques agrégées (jamais de données brutes) pour générer les insights IA. Ces transmissions sont soumises à la politique de confidentialité d'Anthropic.

⚠ La version Pro nécessite une clé API Anthropic valide définie en variable d'environnement ANTHROPIC_API_KEY. Les coûts d'utilisation de l'API sont à la charge de l'opérateur.
📁

Article 4 — Traitement des fichiers uploadés

Art. 4

Les fichiers Excel uploadés sont traités comme suit, quelle que soit la version :

  • Chargé en mémoire vive pour analyse
  • Sauvegardé temporairement dans le dossier uploads/ le temps de l'analyse
  • Supprimé immédiatement après traitement, y compris en cas d'erreur
  • Jamais transmis à un tiers (hors API Claude en version Pro, sous forme agrégée uniquement)

Pour les fichiers dépassant 1 000 lignes, PerfCNC effectue automatiquement un échantillonnage aléatoire de 200 lignes représentatives pour optimiser les performances et les coûts IA. Un avertissement est affiché dans le dashboard.
