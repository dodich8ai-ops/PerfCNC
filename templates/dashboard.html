📋 Total saisies

Définition
Nombre de lignes valides lues dans votre fichier Excel après nettoyage. Chaque ligne représente un poste de production ou une saisie d'opérateur.
Valeur actuelle
Total saisies = {{ total_saisies }} lignes
Utilisation dans PerfCNC
Ce nombre est la base du calcul du temps d'ouverture théorique :
{{ total_saisies }} postes × {{ indicateurs.heures_poste }}h = {{ indicateurs.heures_ouverture_theoriques }}h théoriques

⏱️ Heures produites (moyenne)

Formule
Moyenne = Σ Heures produites / Nombre de saisies
= {{ total_heures_produites }} h / {{ total_saisies }} = {{ moyenne_heures }} h
Interprétation
Une moyenne de {{ moyenne_heures }} h par poste, pour un poste théorique de {{ indicateurs.heures_poste }} h, correspond à une disponibilité brute de {{ indicateurs.disponibilite }} %.

⚠️ Total pièces KO (rebuts)

Formule
Total rebuts = Σ (colonne "Pièces KO")
= {{ total_rebuts }} pièces
Interprétation
Les rebuts sont des pièces non conformes, retirées de la production. {% if indicateurs.taux_qualite is not none %} Le taux qualité de {{ indicateurs.taux_qualite }} % est calculé grâce à la colonne optionnelle « Total pièces ». {% else %} Pour calculer le taux qualité, ajoutez une colonne « Total pièces » dans votre fichier Excel. {% endif %}

📊 TRS — Taux de Rendement Synthétique

Formule générale
TRS = Disponibilité × Performance × Qualité

{% if indicateurs.taux_qualite is not none %} TRS = {{ indicateurs.disponibilite }} % × Perf. × {{ indicateurs.taux_qualite }} %
↳ Performance non mesurable sans cadence nominale dans le fichier {% else %} TRS ≈ Disponibilité = {{ indicateurs.trs }} %
↳ Modèle simplifié : Performance et Qualité non disponibles dans ce fichier {% endif %}
Valeurs de référence industrie
Mauvais <60 % Acceptable 60–75 % Bon 75–85 % Excellent >85 %
Votre TRS actuel : {{ indicateurs.trs }} % {% if indicateurs.trs is not none %} {% if indicateurs.trs >= 85 %}— Excellent ! Niveau classe mondiale. {% elif indicateurs.trs >= 75 %}— Bon niveau. Objectif : franchir les 85 %. {% elif indicateurs.trs >= 60 %}— Zone acceptable. Des actions ciblées peuvent vite améliorer ce score. {% else %}— Zone critique. Analysez les causes d'arrêt et les rebuts en priorité. {% endif %} {% endif %}

🔧 Disponibilité machine

Formule appliquée à vos données
Disponibilité = Heures produites / Heures d'ouverture × 100

= {{ total_heures_produites }} h / ({{ total_saisies }} postes × {{ indicateurs.heures_poste }} h) × 100
= {{ total_heures_produites }} / {{ indicateurs.heures_ouverture_theoriques }} × 100
= {{ indicateurs.disponibilite }} %
Interprétation
Sur les {{ indicateurs.heures_ouverture_theoriques }} h théoriquement disponibles, la machine a réellement produit pendant {{ total_heures_produites }} h. Les {{ (indicateurs.heures_ouverture_theoriques - total_heures_produites) | round(1) }} h restantes correspondent aux arrêts (pannes, changements d'outils, attentes…).
Valeurs de référence
Mauvais <60 % Acceptable 60–75 % Bon 75–85 % Excellent >85 %

🔧 Disponibilité — colonne tableau

Formule par période
Disponibilité = Σ Heures produites / (Nb postes × {{ indicateurs.heures_poste }} h) × 100
Mesure le temps réel de production par rapport au temps théorique d'ouverture. Un arrêt machine, une attente matière ou un changement d'outil réduisent cette valeur.

⚡ Performance — colonne tableau

Formule par période
Performance = Pièces produites / (Heures prod. × Cadence nominale) × 100
{% if trs_timeline.has_cadence %}Calculée avec CADENCE_NOMINALE = {{ trs_timeline.has_cadence }} pièces/heure. Mesure si la machine tourne à pleine vitesse. {% else %}Affiche « — » car CADENCE_NOMINALE n'est pas configuré dans app.py et la colonne « Total pièces » est absente ou CADENCE_NOMINALE = None.{% endif %}

Qualité = (Total pièces − Pièces KO) / Total pièces × 100
